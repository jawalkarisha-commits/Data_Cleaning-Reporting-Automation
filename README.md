# Data Cleaning Analysis & Reporting Automation

# Project Overview:

This project focuses on customer segmentation analysis using data cleaning, reporting automation, and dashboard visualization techniques. The dataset was cleaned using Microsoft Excel and visualized using Power BI to generate meaningful business insights.

# Objective:

The objective of this project is to automate data cleaning and reporting workflows and analyze customer behavior based on age, annual income, and spending score.

# Tools Used:

Microsoft Excel
Microsoft Power BI
CSV Dataset
Dataset Information

# The dataset contains customer information including:

Customer ID
Gender
Age
Annual Income (k$)
Spending Score

# Data Cleaning Process:

Data cleaning was performed using Microsoft Excel.

# Cleaning Steps:
Removed duplicate records
Handled missing values
Standardized inconsistent text values
Verified dataset structure

# The Power BI dashboard includes:

Total Customers
Average Age
Average Income
Average Spending Score
Gender Distribution
Spending Category Analysis
Income vs Spending Analysis

# Key Insights:
Most customers belong to the age group of 20–40 years.
The majority of customers are medium spenders.
High-income customers generally show higher spending scores.
Female customers show slightly higher spending behavior.
Customer segmentation helps businesses improve marketing strategies.
